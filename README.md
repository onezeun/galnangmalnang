## 🛠 사용 기술
| 항목               | 기술/도구                                                                 |
|--------------------|---------------------------------------------------------------------------|
| **프레임워크**     | [Next.js 15](https://nextjs.org/)                                        |
| **언어**           | [TypeScript](https://www.typescriptlang.org/)                            |
| **패키지 매니저**  | [Yarn](https://yarnpkg.com/)                                             |
| **코드 포매팅**    | [ESLint](https://eslint.org/), [Prettier](https://prettier.io/)           |
| **상태 관리**      | [Zustand](https://zustand-demo.pmnd.rs/) (클라이언트), [TanStack Query](https://tanstack.com/query/latest) (서버) |

## ✨ 주요 작업
- Next.js + Supabase 기반 풀스택 구조 설계 및 구현
- React Query 기반 데이터 선 패칭 및 캐싱 구조 구축
- 이미지 프리로드 및 로딩 흐름 최적화 (LCP 60% 단축)
- 관리자 페이지 CRUD 기능 구현
